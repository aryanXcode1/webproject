PROJECT TITLE

Design and Development of a Personal Portfolio Website Using HTML, CSS and JavaScript

OBJECTIVE OF THE PROJECT

The objective of this project is to understand and apply the basic concepts of web development.
This project helps in learning how to use HTML for structure, CSS for styling, and JavaScript for adding interactivity.
The aim is to create a simple, responsive, and informative personal portfolio website.

PROJECT DESCRIPTION

This project is a static personal portfolio website developed using HTML, CSS, and JavaScript.
No external frameworks or libraries are used in this project.

The website includes personal information, skills, certifications, and contact details.
It is designed at a first-year, first-semester academic level and follows basic web development principles.

TECHNOLOGIES USED

HTML5 – Used for creating the structure of the website
CSS3 – Used for styling, layout design, responsiveness, and dark mode
JavaScript – Used for dynamic greeting, live time display, dark mode toggle, and form validation

FEATURES OF THE WEBSITE

Responsive webpage design
Fixed navigation bar
Time-based greeting using JavaScript
Live digital clock
Dark mode functionality
Skills section
Certificates section with verification links
Contact form with basic validation

JAVASCRIPT FUNCTIONALITIES

AI Greeting Function
Displays greeting messages like Good Morning, Good Afternoon, etc., based on the current system time.

Live Time Function
Displays the current time and updates every second.

Dark Mode Toggle
Allows the user to switch between light mode and dark mode.

Form Validation
Checks whether name and email fields are filled before submitting the form.
